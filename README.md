## ⚡ خطوات الحصول على APK

---

### الخطوة 1 — حساب GitHub مجاني
اذهب إلى: **github.com/signup**
سجّل بالإيميل — مجاني 100٪

---

### الخطوة 2 — Repository جديد
1. اضغط **+** في الأعلى
2. اختر **New repository**
3. الاسم: `phytoinspect`
4. اختر **Public**
5. اضغط **Create repository**

---

### الخطوة 3 — رفع الملفات
1. في صفحة الـ repository اضغط **uploading an existing file**
2. ارفع كل محتوى هذا المجلد كما هو
3. تأكد أن مجلد `.github` مرفوع أيضاً
4. اضغط **Commit changes**

---

### الخطوة 4 — انتظر البناء ⚙️
1. اضغط تبويب **Actions** في الأعلى
2. ستجد **🌿 بناء PhytoInspect APK** يعمل تلقائياً
3. انتظر **8-15 دقيقة** حتى يتحول للأخضر ✅

---

### الخطوة 5 — تحميل APK
1. اضغط على الـ workflow الأخضر
2. انزل لأسفل ← قسم **Artifacts**
3. اضغط **PhytoInspect-v7-APK**
4. ينزل ملف ZIP → افتحه → ستجد **app-release.apk**

---

### الخطوة 6 — تثبيت على M35 5G

#### أولاً — تفعيل المصادر غير المعروفة:

**عربي / Français:**

الإعدادات / Paramètres ⚙️
← التطبيقات / Applications
← ⋮ النقاط الثلاث
← الوصول الخاص / Accès spécial
← تثبيت تطبيقات غير معروفة / Installer des applis inconnues
← Chrome ← **فعّل / Activer**

#### ثانياً — التثبيت:
1. انقل `app-release.apk` للهاتف عبر USB أو Google Drive
2. افتح الملف من هاتفك
3. اضغط **Installer**
4. اضغط **Ouvrir** بعد الانتهاء

---

### الخطوة 7 — إعداد مفتاح Gemini API

1. افتح: **aistudio.google.com/apikey**
2. سجّل بحساب Google
3. اضغط **Create API Key**
4. انسخ المفتاح (`AIza...`)
5. ألصقه في التطبيق عند أول تشغيل

---
